# RBMS 부품 목록 (Bill of Materials)

ESP32-C6 기반 파충류 사육장 모니터링 시스템의 전체 부품 목록.
Type A(콘센트 전원, Router)와 Type B(배터리 전원, SED) 노드별로 구분한다.

---

## Type A 노드 (Mains Power, Thread Router)

콘센트 전원 공급, FreeRTOS 멀티태스크, SSR/PWM 액추에이터 제어 노드.

### MCU 및 전원

| # | 부품명 | 수량 | 패키지 | 주요 사양 | 비고 |
|---|--------|------|--------|-----------|------|
| A1 | ESP32-C6-WROOM-1 | 1 | Module (18x20mm) | RISC-V 160MHz, 512KB SRAM, 4MB Flash, Wi-Fi 6 + BLE 5 + 802.15.4 (Thread) | Espressif. 안테나 내장 모듈, PCB 안테나 클리어런스 확보 필요 |
| A2 | AMS1117-3.3 | 1 | SOT-223 | LDO 3.3V, 1A, Dropout 1.1V (typ.), Vin 최대 15V | USB 5V -> 3.3V 변환. 입력 10uF + 출력 22uF 이상 권장 |
| A3 | USB-C 커넥터 | 1 | SMD 16-pin | USB 2.0 Type-C 리셉터클, 5V 전원 전용 | CC 핀에 5.1k 저항 2개 (CC1, CC2 -> GND) 필수 |

| A5 | DS18B20 방수 프로브 | 2 | 스테인리스 프로브 + 케이블 | 1-Wire 온도, -55~125C, 정확도 +/-0.5C (12-bit) | Analog Devices/Maxim. 프로브 #0=핫존, #1=쿨존. ROM 코드로 자동 식별 |

> 프로토타입 단계에서는 AliExpress + 국내 전자부품몰 조합이 비용 효율적이다.
